1. Introduction

In this project, you will study the dynamic branch prediction on SimpleScalar. The project consists of three tasks, as listed below:

  • In the first task, you will experiment with a built-in 2-bit branch predictor on SimpleScalar and use the compress benchmark program to evaluate its prediction accuracy.
  • In the second task, you will implement a 3-bit branch predictor on SimpleScalar and evaluate its prediction accuracy using the compress benchmark.
  • In the third task, you will design and implement your own branch predictor on SimpleScalar and try to beat the built-in predictors in terms of prediction accuracy using the compress benchmark.

Each group should submit ONE set of code and report with names of all group members.

3. Grading

  • Evaluation of the built-in 2-bit predictor (20%)
  • Implementation and evaluation of the 3-bit predictor (30%)
  • Design, implmentation, evaluation of your own predictor (40%)
  • Quality of project report (10%).
You should include the evaluation results and describe the design of your own predictor in the project report. If you have referenced any books, websites, or classmates, please acknowledge them explicitly in your report. Otherwise, it may be considered as cheating.

4. Submission

Please create a directory "comp4611_proj_groupID" to save the code and report files. In specific, please save your code file for 3-bit predictor "bpred.c" under the sub-directory “comp4611_proj_groupID/3bit/”, save the code files for your own predictor, including bpred.h, bpred.c, sim-bpred.c, readme and other relevant files, under the sub-directory “comp4611_proj_groupID/own/”, save the report file as "report.doc" under the directory "comp4611_proj_groupID/", then package the directory "comp4611_proj_groupID" in one zip file, name it as “comp4611_proj_groupID.zip”, and submit it to CASS (https://course.cs.ust.hk/cass). The report file "report.doc" should include all documentation for the three tasks, and your group ID, each group member’s name, UST ID, email on the first page. You must also submit a hard copy of your project report to the homework box for COMP4611. Output trace files for 3-bit predictor and your own predictor will be submitted separately (not CASS).

Only one of the group members submits the files and the hard copy of the report. (Submiting multiple copies by multiple group members is not allowed.) Both the hard copy submission and electronic submission need to be submitted before the due date and time. NO late submission will be accepted! If either part of the submission (hard copy or electronic submission) is missing, the project will not be graded and no points will be given.
